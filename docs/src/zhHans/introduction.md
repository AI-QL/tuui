# 导言

**TUUI** 是基于 MCP（模型上下文协议）的 AI 工具整合应用，用于开发智能跨平台桌面应用。它使用 `Vue 3`，使您能够轻松构建快速的开发环境。

## 使用优势

- ✅ 无需任何预设，即可立即构建，快速开发。
- ✅ 快速维护，与最新的 `Vue` 和 `Electron` 以及许多模块兼容。
- ✅ 通过使用各种附加模板，无需担心布局和数据管理。

## 特点

- ✨ 通过 MCP 协议快速对接 AI 工具
- ✨ 动态管理不同厂商的语言模型后端 API
- ✨ 跨平台开发和构建支持
- ✨ 支持自动化应用程序测试
- ✨ 支持 TypeScript
- ✨ 多语言支持
- ✨ 基本布局管理器
- ✨ 通过 Pinia 存储进行全局状态管理
- ✨ 通过 GitHub 社区和官方文档提供快速支持
